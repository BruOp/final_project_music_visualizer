## Getting Started

1. `bundle install` 
2. Ensure you have the latest PhantomJS version from
    https://github.com/teampoltergeist/poltergeist#installing-phantomjs
2. `shotgun -p 3000 -o 0.0.0.0`
3. Visit `http://localhost:3000/` in your browser
